# watamelon Discord Bot

## Usage
LINUX:<br>
`chmod +x start_server.sh`<br>
`./start_server.sh`<br>

## Installation
Create a python virtual environment (venv):<br>
`python3 -m venv .venv`<br>

### Bash
Then, in a terminal of your choice, enter your venv environment then pip install the requirements.txt:<br>
`source .venv/bin/activate`<br>
`pip install -r requirements.txt`<br>

### .env
Include a .env file in the root of your folder.<br>

Example: <br>
DISCORD_TOKEN=<br>
OWNER_ID=<br>
BOT_ID=<br>
TEXTCHANNEL=<br>
GUILD=<br>
